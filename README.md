### ***Description***
#### A "Battleship" redux as a browser game. Instead of "maritime war simulation, for kids!", the intention is "competitive positivity." 

#### General Assembly SEI Unit 1 project.
***

### ***Technologies***
* HTML
* CSS
* JavaScript
